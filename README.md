# ニキマスク / NIKIMASKU
**Private Mask Painter**

> 「引かぬ。媚びぬ。ナンバーは消す。」

車の写真をSNSに上げるとき、ナンバープレートや人の顔を
サクッと隠せるプライバシー保護ツール。

ブラウザだけで動く。インストール不要。
撮った写真はこの端末の外に一切出ない。

---

## できること

- ナンバープレートを **白く塗りつぶす**（自由変形対応・斜め撮りもOK）
- 人の顔を **ソフトボカシ** で自然に隠す
- 処理した箇所は **コピーして複数箇所に展開**
- iOS Safari / Android Chrome 対応

---

## 使い方

1. 「写真」ボタンで画像を読み込む
2. 隠したい箇所をタップ → 枠を調整
3. 「白く塗る」または「ボカす」で確定
4. 「保存」で完了
